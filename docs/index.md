## What is Ariadne?

Intelligence about a single entity — a person, a unit, an organizational node —
is scattered across heterogeneous systems and modalities. No one query interface
spans that range, so analysts pivot manually between tools, losing context at
every step. Ariadne uses the **Claude Agent SDK** as a unifying analytic
interface: an orchestration layer that dispatches specialized tools across graph,
structured, and unstructured sources in one coordinated, multi-hop workflow —
then synthesizes the findings into a cited analytic product.

## The hard part isn't access — it's reasoning

The decisive link between two facts may exist only through an *implicit*
organizational relationship, buried in a different store and a different
modality. Ariadne is built to follow that thread.
